# baby-log-app

A simple web app for logging and tracking a baby's day-to-day activities (feeds, sleep, diapers, and more), designed to be shared between two parents.

## Goals

- **Web-based** — accessible from any phone browser, no app install required.
- **Shared account** — both partners sign in and see the same private data in real time.
- **Simple** — quick to log an event in a few taps.
